## Additional Notes

### Python Virtual Environment
- Python version: 3.14
- Create with: `python3 -m venv .venv && source .venv/bin/activate`
- Install deps: `pip install -r backend/requirements.txt`

### Node Dependencies
- Frontend: `cd frontend && npm install`
- GSOC: `cd gsoc && npm install`

### Database
- Dev uses SQLite by default (`sqlite:///./goldensignals.db`)
- PostgreSQL available for staging/production
- Redis on default `localhost:6379`

### Ports
| Service          | Port |
|-----------------|------|
| Backend API     | 8000 |
| WebSocket       | 8001 |
| MCP Server      | 8002 |
| Frontend (Next) | 3000 |
| PostgreSQL      | 5432 |
| Redis           | 6379 |

### Keys to Rotate Before Production
- All `SECRET_KEY` / `JWT_SECRET_KEY` values
- All API keys (OpenAI, Anthropic, Grok, Google, Perplexity, etc.)
- Tradier API key + account number
- Kalshi credentials
- Discord bot token
- Twitter API keys
- News API key
- All data provider keys (TwelveData, Finnhub, Polygon, Alpha Vantage, FMP)
