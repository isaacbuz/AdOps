# Paid Media Acquisition Workflow

1. Wait for Strategy & Creative handoff.
2. Verify the channel of the incoming ticket.
3. If channel is O&O, expect new placements tickets.
4. If channel is Display/Programmatic, expect new placements, creative rotation, new campaigns, placement name change, URL change.
5. If channel is CTV, expect new placements, new campaigns, URL change, placement name change.
6. If channel is Social Media, expect new placements, creative rotations, new campaigns, URL change, placement name change.
7. If channel is Search, expect new campaigns and new placements tickets.
8. Submit tickets to PADOPS Jira board (or Airtable).
9. Follow Trafficking 101 instructions to process tickets.
10. Send tags back to the media channel reporter.
