# MLP Troubleshooting Process

1. Receive ticket regarding identified issue with URL & contextual information.
2. Recreate the issue for the Web Dev team.
3. Create an escalation ticket for the MLP Team.
4. Wait for MLP team to fix the issue and close the ticket.
5. Notify Ad Ops and stakeholders of the solution.
