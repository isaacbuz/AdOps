# EVE YouTube Ticket Workflow

1. Send to BOAT from the #4 base or create a net-new ticket in the New BOAT Ticket Form (NBTF).
2. Set the "Platform" to YouTube.
3. Select the target audience in "Audience Tactic(s) [Reporter-ID'ed]" (e.g. "YouTube - A18+").
4. Select the "Campaign Name" and verify Content Type, Brand, and Title.
5. Populate the Excel CSV file template using the guide (ensure columns A through L, N, and O are filled). Ensure the YouTube Video ID is provided.
6. Save the CSV file with UTF-8 encoding ("CSV UTF-8 (Comma delimited) (.csv)").
7. Upload the CSV file as an attachment and submit the NBTF.
8. Change BOAT ticket Status from "Backlogged" to "In Queue".
9. Click "Launch EVE" to open the EVE Pre-trafficking Page (EPreP).
10. Verify Airtable parameters, upload the assets in EPreP, and click "BUILD PLATFORM DATA".
11. Confirm any platform override IDs (including YT Ad Group ID) and click "Mark as Ready to Traffic".
12. Once Status is "Done", QA the DV360/YouTube end results, making sure to append the companion banner to the EVE-created DV360 Ads.
